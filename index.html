NC News

A single-page app hosting user-submitted stories, allowing for users to add comments and give points to both the articles and comments. All submitted data is hosted on a seperate API.

This project was completed as part of the Northcoders Bootcamp. Both the back- and frontend were the final, week-long tasks done as a summary to the respective sections of the course.

Backend: PostgreSQL, Express
Github Repo Hosted API

Frontend: React
Github Repo Hosted App

StandingsSlider

A single-page app for comparing historical data for Formula 1 seasons and drivers since 1960.

When browsing seasons or drivers by points, the scoring system can be changed to any that has been used for Formula 1 since 1960; data is recalculated to display how a season or career would turn out under the selected system.

Backend: MySQL, Express
Github Repo Hosted API

Frontend: Vue
Github Repo Hosted App
